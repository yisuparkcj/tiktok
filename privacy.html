개인정보처리방침

본 애플리케이션은 TikTok API를 통해 수집된 정보를 외부에 저장하거나 제3자와 공유하지 않으며, 사용자 본인의 TikTok 계정으로부터 직접 인증된 범위 내에서만 데이터를 조회합니다.

1. 수집하는 정보

OAuth 인증을 통해 TikTok 계정에 접근하며, 사용자가 동의한 범위 내에서 영상 통계 정보(조회수, 좋아요 수 등)를 가져옵니다. 사용자 식별 정보(ID 등)는 저장되지 않습니다.

2. 정보의 이용

수집된 정보는 단순히 사용자에게 해당 TikTok 계정의 콘텐츠 통계를 제공하는 용도로만 사용되며, 서버에 저장되지 않습니다.

3. 정보 보관 및 공유

본 애플리케이션은 수집된 데이터를 저장하거나 제3자에게 제공하지 않습니다.

4. 책임

OAuth 인증은 TikTok의 보안 프로토콜을 따르며, 애플리케이션은 사용자 정보에 대해 별도의 저장, 분석, 광고 목적의 활용을 하지 않습니다.

5. 문의

개인정보와 관련한 문의사항은 GitHub 리포지토리 내 Issue 기능 또는 등록된 이메일을 통해 문의하실 수 있습니다.
